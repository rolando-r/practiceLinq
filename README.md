## Run Locally

Clone the project

```bash
  git clone https://github.com/rolando-r/practiceLinq/
```

Go to the project directory

```bash
  cd practiceLinq
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
